# ChatGPT Clone (Frontend)

A **ChatGPT-style frontend application** built using **React.js** and **Material UI**.  
This project focuses purely on the **user interface and client-side behavior** of a chat application inspired by ChatGPT. There is **no backend** included in this repository.

The application runs locally on **port 5173**.

---

## Overview

This project demonstrates how to build a modern conversational UI using React and Material UI. It can be used as:
- A UI prototype for AI chat applications
- A frontend practice project for React + MUI

## Features

- ChatGPT-like conversational interface  
- Built with **React.js**  
- Styled using **Material UI (MUI)**  
- Component-based, reusable UI structure  
- Runs entirely on the frontend  
- No backend or database dependency  

---

## Tech Stack

- **React.js** – Frontend framework  
- **Material UI (MUI)** – UI components and styling  
- **Vite** – Development server and build tool  

---

## Application Type

- **Frontend only**
- No backend server
- No API integration by default
- No authentication or persistence
